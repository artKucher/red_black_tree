# red_black_tree
Red black tree on C#
Задание
  Постройте красно-черное дерево по заданным ключам Vector(53, 38, 65, 98, 38, 76, 4, 28, 14, 33, 25, 21, 54, 58, 32, 25, 63, 63,     94, 1, 86)
  Реализуйте красно-черное дерево со следующими операциями:
    создание дерева
    добавление элемента
    поиск элемента в дереве
